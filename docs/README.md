# IBM Geospatial Exploration and Orchestration Studio Docs

This section has the documentation for the IBM Geospatial Exploration and Orchestration Studio.

## To update docs page

1. Install [hatch](https://hatch.pypa.io/1.9/install/#command-line-installer) in your workstation
2. Make changes
3. To serve locally run `npm run docs-serve` and to build run `npm run docs-build`
4. Commit and push changes to github
5. Run the following command: `npm run docs-deploy` and it will push github pages site.

## Live preview

### From the terminal

```bash
npm run docs-serve
```

NB: Run all commands from the root directory of the repo
